# PixelNest

PixelNest is a beautiful and immersive wallpaper discovery application built with React Native and Expo. Browse, search, and download high-quality images from a vast collection.

## Features

- **Discover**: Browse a curated feed of high-quality pictures.
- **Search**: Find the perfect image with a powerful search feature.
- **Categories**: Explore images by categories (Nature default).
- **Favorites**: Like and save your favorite pictures to a local collection.
- **Download**: Download full-resolution images directly to your device.
- **Smooth UI**: Built with Reanimated for fluid interactions and transitions.

## Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Routing**: [Expo Router](https://docs.expo.dev/router/introduction/)
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)
- **List Performance**: [@shopify/flash-list](https://shopify.github.io/flash-list/)
- **Image Handling**: [Expo Image](https://docs.expo.dev/versions/latest/sdk/image/)
- **Styling**: StyleSheet with custom theme constants

## Installation

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd Wallpie_React_Native
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the app**:
    ```bash
    npm start
    ```

4.  **Run on device/emulator**:
    -   Press `a` for Android Emulator
    -   Press `i` for iOS Simulator
    -   Scan the QR code with Expo Go on your physical device

## Project Structure

- `app/`: Expo Router pages and layouts
- `components/`: Reusable UI components
- `constants/`: Theme, data, and configuration
- `helpers/`: Utility functions and API helpers
- `assets/`: Static assets like images and fonts
